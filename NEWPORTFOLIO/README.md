# ✧ DEEPAK SINGH PORTE — Modern Motion Portfolio

A premium, motion-focused developer portfolio designed with a focus on cinematic aesthetics, high-performance interactions, and minimalist design.

![Portfolio Preview](https://images.unsplash.com/photo-1545665277-5937489579f2?auto=format&fit=crop&q=80&w=1200)

## ✧ Vision

This project bridges the gap between functional engineering and digital art. Every interaction—from the liquid cursor to the staggered text reveals—is choreographed to create a premium user experience that feels alive and responsive.

## 🚀 Key Features

- 🪩 **Liquid Movement**: A custom-engineered magnetic cursor with spring physics for a fluid, organic feel.
- 🎭 **Cinematic Reveals**: Scroll-triggered text animations using Framer Motion with custom cubic-bezier easings.
- ⚡ **Performance First**: Optimized 60FPS animations, GPU-accelerated transforms, and Zero Layout Shift (CLS-safe).
- 🧬 **Monorepo Architecture**: A unified structure for React/Vite frontend and Express/Node backend.
- 📱 **Responsive Precision**: Fully adaptive layouts from 8K desktops down to mobile viewports.
- 🌊 **Smooth Scroll**: Integrated Lenis for a luxurious scrolling experience.

## 🛠 Tech Stack

- **Frontend**: [React.js](https://reactjs.org/), [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Motion**: [Framer Motion](https://www.framer.com/motion/), [GSAP](https://greensock.com/gsap/)
- **Backend**: [Node.js](https://nodejs.org/), [Express](https://expressjs.com/)
- **Database**: [MongoDB](https://www.mongodb.com/) (with Fallback Engine)
- **Deployment**: [Vercel](https://vercel.com/)

## 📂 Structure

## ⚙️ Local Development

1. **Clone the repository**:

   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd NEWPORTFOLIO
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Run Services**:
   - Start Backend: `npm run dev:server` (Port 5000)
   - Start Frontend: `npm run dev:client` (Port 5173)
